### 留言板(支持增删查)nodejs实现

#### 目录介绍

​	1.app.js(程序主入口)

​	2.models(业务模块,封装的数据库基础操作模块)

​	3.views(ejs模板引擎存放)

​	4.dbset.js设置数据库连接信息

​	5.public(存放前端静态资源bootstrap,jquery,underscore等资源)

#### 使用说明

​	1.自行去安装MongoDB(版本需要大于3.0)

​	2.使用案例前请先安装依赖

```
 npm install
```

​	3.启动项目

```
node app.js
```
